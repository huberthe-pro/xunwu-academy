# 寻吾书院 (Xunwu Academy) - 产品文档

## 1. 产品愿景
「寻吾书院」致力于打造一个充满中国古墨韵味的文化分享与管理平台。平台以前端雅致的视效展现中国传统文化之美（如国学讲堂、历代金石、雅集拾音等），并以后端严谨的“书院规制”来维护内容生态。

## 2. 核心受众
- **文化爱好者与研习者**：前台浏览者，寻求高质量传统文化内容的学习与阅览。
- **书院执笔者（内容管理者）**：后台维护者，负责发布、归档、整理平台内容。

## 3. 功能架构

### 3.1 前端展示 (Frontend)
- **水墨视觉风格**：全站采用低饱和度水墨背景、竖排版印章（Seal Red）、古典渐变与毛玻璃质感，营造“深山古刹”般静谧的阅读体验。
- **内容矩阵**：
  - **总导航（讲堂、修藏、驿站）**：清晰划分文化类别。
  - **动态组件**：如频道卡片悬浮时的墨迹晕染（Ink Spread）效果，文章列表的雅致排版。

### 3.2 书院后台 (Admin Dashboard)
- **总览仪表盘**：全站数据监控（文章总数、频道分类、过客足迹）。
- **内容管理（卷帙修撰）**：支持文章的长文本结构化存储，涵盖新建、修撰、删除与实时预览。
  - **富文本图文排版**：全面接入 Tiptap 模块化编辑器，支持多级标题、列表、文本格式与段落控制。
  - **多媒体云盘**：内置 Supabase Storage 图片上传机制，支持本地文件限制与云端链接直插。
  - **阅量控制**：前台访客浏览自动增加阅量，后台也提供灵活的人工数据干预入口。
- **频道管理（门类开辟）**：灵活增删改查前端的分类节点。
- **系统设置与身份管理**：
  - **多账号与权限隔离（RBAC）**：支持“超级执笔者”与普通“执笔者”。普通执笔者仅能管理内容，不可见底层架构配置。
  - **行迹微录（System Logs）**：对所有关键操作（新建内容、删除账号等）进行不可篡改的留痕，支持跨端跨日期的日志回溯（当前基于本地沙盒，后期可无缝接入云端数据库）。

## 4. 体验亮点
- **沉浸式登堂入室**：`/login` 采用专属的名讳与印信进行校验。
- **去系统化设计**：将西式的“Admin”概念中式本地化，用“执笔者”、“卷帙”、“行迹微录”等概念替代干瘪的 IT 术语，使体验完整沉浸。
